Digital Invoice Tracking and Management System

A full-stack web application that helps organizations track, manage, and process invoices digitally.
The system allows users to add invoices, monitor status, mark invoices as completed, and view records stored in the database.

🚀Features
  Add new invoice records
  View invoice list
  Track invoice status
  Mark invoice as Completed
  Store and manage invoice data in database
  Clean and responsive UI
  Backend API for invoice processing
